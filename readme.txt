Calculator App created by Jeffrey Wang A01228943

Contains multiple modules for multiple parts of the calculator.
The buttons and display screen are in different files, and the functions for the calculator are in App.js
There is a CSS file that contains the style components to the App file. You are able to perform standard calculations on this calculator.
When buttons are clicked, the display changes to output, and once the = button is clicked, the resuilts of the display is calculated. 
C button will slice the last character in the display
CE button will clear the display
